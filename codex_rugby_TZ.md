# 🏉 ТЗ для Codex: сайт про регби (пошаговые промпты)

## Как использовать
Копируй промпты по очереди в Codex и выполняй шаг за шагом.

---

## 1. Общие правила

```
Ты помогаешь мне делать production-ready MVP сайта о регби для украинской аудитории.

Требования:
- Next.js App Router
- TypeScript
- Tailwind CSS
- украинский язык интерфейса
- чистый адаптивный дизайн
- не усложнять без необходимости

После каждого задания:
1) перечисляй файлы
2) объясняй как запустить
3) не выходи за рамки задания
```

---

## 2. Создание проекта

```
Создай Next.js проект с App Router, TypeScript и Tailwind.

Нужно:
- структура проекта
- layout
- header/footer
- страницы:
  - Головна
  - Новини
  - Чемпіонати
  - Команди
  - Гравці
  - Для новачків
  - Про проєкт

Покажи структуру и запуск.
```
