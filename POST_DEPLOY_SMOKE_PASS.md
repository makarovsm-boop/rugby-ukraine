# Post-Deploy Smoke Pass

Короткий ручний smoke pass одразу після deploy.

Мета:

- швидко зрозуміти, що MVP живий
- відловити критичні поломки до того, як сайт почнуть активно відкривати

## 1. Home

Відкрити:

- `/`

Перевірити:

- сторінка відкривається без server error
- шапка працює
- переходи на `Новини`, `Матчі`, `Для новачків` працюють
- перший екран виглядає цілісно
- немає зламаних картинок

## 2. Auth

Відкрити:

- `/login`

Перевірити:

- форма логіну відкривається
- логін працює
- логаут працює
- `callbackUrl` працює правильно
- `USER` не має доступу до адмінки
- `EDITOR` бачить лише статті і коментарі
- `ADMIN` має повний доступ

## 3. Admin

Відкрити:

- `/admin`
- `/admin/articles`
- `/admin/comments`
- `/admin/users`

Перевірити:

- `ADMIN` бачить адмінку без помилок
- `EDITOR` не отримує зайвий доступ
- можна відкрити сторінку створення/редагування статті
- можна відкрити moderation comments
- форми не ламаються

## 4. Robots

Відкрити:

- `/robots.txt`

Перевірити:

- файл доступний публічно
- у ньому є `Sitemap`
- `/admin` не індексується

## 5. Sitemap

Відкрити:

- `/sitemap.xml`

Перевірити:

- файл доступний
- є основні публічні сторінки
- немає `/search`

## 6. Search

Відкрити:

- `/search?q=Україна`
- `/search?q=Leinster`

Перевірити:

- пошук видає результати
- тип сутності показується коректно
- переходи в результатах працюють
- порожній стан не виглядає зламаним

## 7. Comments

Відкрити:

- будь-яку новину, наприклад `/news/ukraine-squad-summer-camp`

Перевірити:

- блок коментарів відображається
- неавторизований користувач бачить заклик увійти
- авторизований користувач може відкрити форму коментаря
- список коментарів не ламає верстку

## 8. Minimum Route Set

Якщо часу мало, мінімальний обов'язковий набір:

- `/`
- `/news`
- `/matches`
- `/login`
- `/admin/articles`
- `/search?q=Україна`
- `/robots.txt`
- `/sitemap.xml`

## 9. Blocking Issues

Вважати критичними:

- 500/error screen на ключових сторінках
- неробочий логін
- неробоча адмінка для `ADMIN`
- недоступні `robots.txt` або `sitemap.xml`
- повністю зламаний пошук
- повністю зламані коментарі на статті

## 10. Smoke Pass Verdict

Після deploy MVP можна вважати базово живим, якщо одночасно:

1. відкривається головна
2. працює логін
3. `ADMIN` заходить в адмінку
4. `robots.txt` і `sitemap.xml` доступні
5. пошук повертає результати
6. сторінка статті з коментарями не ламається
